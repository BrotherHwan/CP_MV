# CP_MV
## 프로젝트 목적
AI모델인 Pose estimation을 활용하여 자세교정 프로그램을 구현합니다.<br/> 
## 개발환경
VScode, Linux(Ubuntu)
## 수행역할
관절 각도 연산과 유사도에 기반한 점수 산출 알고리즘을 구현.
